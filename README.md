# Taskboard

Personal task manager with Kanban boards, sortable list view, and a metrics dashboard.

## Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Vite + React + TypeScript + Tailwind CSS + shadcn/ui |
| Backend | Hono + Node.js + TypeScript |
| ORM | Drizzle |
| Database | PostgreSQL |

- Docker (for PostgreSQL)
