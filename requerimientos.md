Aplicación para gestionar tareas.

Tiene que poder añadir, modificar y eliminar tareas.

La aplicación tendrá como mínimo dos vistas:

1. Una para ver las tareas a modo de lista y que se puedan ordenar por todas las columnas.
2. Tableros Kanban

Al añadir tarea nueva, se le tendrá que indicar a qué tablero va.

Existirá un tablero Kanban por defecto, que es el que se indicará a crear una tarea predeterminadamente, pero se podrán crear tableros adicionales.

Estos tableros Kanban por defecto tendrán las siguientes columnas:

* Pendiente
* En progreso
* Parada
* Finalizada

A un tablero concreto se le pueden añadir más columnas, es decir, los tableros serán configurables.

Las tareas serán los siguientes campos:

* Título
* Descripción
* Fecha inicio
* Fecha fin

Todos obligatorios, menos las fechas.

Las tareas se podrán mover de columna a columna del tablero arrastrándolas.
